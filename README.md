# Kotlin Sandbox

## About

Kotlin Sandbox is my personal learning project for exploring Android development with Kotlin and Jetpack Compose. This repository documents my progress as I work through the Android Basics course, experiment with new concepts, and apply what I learn by building small features and examples.

## Goals

* Learn Kotlin and Jetpack Compose.
* Practice Android development through hands-on projects.
* Document my progress and experiments.
* Build a strong foundation for creating real-world Android applications.

## Note

This repository is a learning sandbox, so the code, structure, and features will continue to evolve as I gain more experience.

*I hope this journey helps me grow into a skilled Android developer.*
